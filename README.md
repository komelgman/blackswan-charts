# blackswan-charts

Monorepo layout:
- `packages/charts-lib`: charting library
- `apps/charts-demo`: demo application using the library

## Setup

```sh
npm install
```

## Development (demo app)

```sh
npm run dev
```

## Build

```sh
npm run build:lib
npm run build:demo
```

## Tests

```sh
npm run test:unit
npm run test:ct
npm run test:e2e
```
