# @blackswan/context-menu

Reusable context-menu components, directive, and types.

## CSS Variables

This package relies on the following CSS variables for styling:

- `--chart-text-color`
- `--chart-font`
- `--menu-background-color`
- `--menu-background-color-on-hover`

If you use this package outside of the charts context, define these variables in your consumer app (for example, on a wrapper element or `:root`).
